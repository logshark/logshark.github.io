SystemVerilog - Event

2023-06-25
#SystemVerilog

Event是靜態物件，用來同步兩個或更多個運行中的process，一個process會觸發Event，其他process等待(wait)這個event。

Event的宣告方式

Event的使用方式:

觸發Event: 用 -> 或 ->> 運算元
等待Event: 用 @ 運算元 或 .triggered
